# GithubIntegration

- node version 18.20.x

- install node_modules

```sh
  npm install
```

- create .env in in github-integration from .env.example

- server run command

```sh
  npm run dev
```

# GithubIntegrationFrontend

- node version 18.20.x
- ng version 18.2.0

- install node_modules

```sh
  npm install
```

- server run command

```sh
  ng serve
```
